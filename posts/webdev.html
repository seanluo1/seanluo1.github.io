Learning Website Development

July 27, 2018

College Park, Maryland

I started learning web development in my free time this past summer (2018). I've always had an interest in web dev, and I figured this summer was the perfect time to learn CSS, HTML, and Javascript since I was going to begin applying to a variety of internships that required different skills.

Thanks to resources from Lynda.com, I was able to learn basic concepts from the three aforementioned markup and scripting languages. This website is the direct result of those lessons, as I wrote every line of code that makes this website appear and function how it does from scratch. No Bootstrap, Node.js, or Sass (although I want to eventually learn about these as well), everything was written starting from blank text documents.

I absolutely loved creating this website. Writing HTML and CSS felt more like being a designer than a programmer and flexed my creative muscles in different way than my other programming experiences have done in the past.

I learned how to structure a website properly with parent and child sections, how to mobilize a website to make it loook better on mobile devices, and how to host multiple pages on a website.

Web development has been a lot of fun so far. I'm excited to learn more and to begin using more advanced tools such as Sass and Node.js. I still have a lot of work to get done on this website before I'm happy with how it looks, but for now I'm glad that I was able to teach myself such a fun and useful skill.
